
Identity Management Maturity – ISO 27001:2022 (Controle 5.16)

Objetivo

Modelo educacional para avaliação da maturidade do ciclo de vida de identidades (Identity Lifecycle Management), baseado na ISO/IEC 27001:2022 – Controle 5.16.

Escopo

Este módulo avalia:

Onboarding (Joiner)

Movimentação (Mover)

Offboarding (Leaver)

Integração com RH

SLA de desprovisionamento

Gestão de contas não humanas

Rastreabilidade

Modelo de Maturidade

0 – Inexistente
1 – Ad-hoc
2 – Documentado
3 – Padronizado
4 – Automatizado
5 – Baseado em Risco

Como Executar

Abra o arquivo index.html localmente no navegador.

Não requer backend ou banco de dados.

Observação

Projeto educacional e independente.
Baseado exclusivamente em normas públicas.
